# Project 2: Exploratory Data Analysis (EDA)

## Objective
Analyze a cleaned dataset to identify trends, patterns, and business insights.

## Tasks Performed
- Calculated Mean, Median, and Count
- Identified Best-Selling Products
- Analyzed Payment Methods
- Evaluated Order Status Distribution
- Determined Highest Revenue Product

## Tools Used
- Microsoft Excel
- Pivot Tables

## Key Findings
- Chair was the best-selling product (562 units sold)
- Online was the most used payment method (258 orders)
- Cancelled orders were the most common status (250 orders)
- Chair generated the highest revenue (₦195,620.11)
