# Project 4: Data Visualization with Power BI

## Objective
The objective of this project was to transform sales data into meaningful visual insights using Microsoft Power BI. The dashboard was designed to help stakeholders understand key business metrics and make data-driven decisions.

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dashboard Features
- KPI Cards
  - Total Revenue
  - Total Orders
  - Total Units Sold
  - Average Unit Price
- Best Selling Products
- Revenue by Product
- Payment Method Distribution
- Order Status Distribution
- Interactive Slicers

## Skills Demonstrated
- Data Visualization
- Dashboard Design
- DAX Measures
- Business Intelligence
- Data Storytelling
- Data Analysis

## Key Insights
- Chair was the best-selling product based on quantity sold.
- Chair generated the highest revenue.
- Online payments were the most frequently used payment method.
- Order status analysis revealed opportunities for operational improvements.

## Project Outcome
This dashboard provides an interactive view of sales performance, enabling users to filter data and gain valuable business insights quickly.
